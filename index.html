Projects

Zombie Outbreak Survival Game

Zombie Outbreak: Survival is a zombie survival game where you must survive as long as you can while fending off the hordes of the undead. The game has straightforward gameplay and high-quality graphics. The code for the game is clean and understandable, following best practices such as modular design, meaningful naming, and consistent formatting. Comments and documentation are included to explain complex logic. Blueprint Interfaces are used for communication between actors, and duplicate code is avoided by creating reusable functions. Assets are organized in the Unreal Editor to improve maintainability, and complex nesting is avoided. Regular testing and refactoring help optimize performance and maintain a tidy codebase. Note: The Game is UnderDevelopment will be added in itch.io in future.

2D Space Game

In the development of the 2D shooter game "Space Runaway" using Unity and C#, I emphasized maintaining a clean and understandable codebase with a focus on reusability. To achieve this, I employed a modular design approach, breaking down the project into smaller, self-contained components for player movement, enemy AI, weapon systems, and level generation. Descriptive naming and thorough comments were used throughout the code and blueprints, ensuring clarity and facilitating comprehension for both current and future developers. Additionally, I leveraged Scriptable Objects to manage game data and configurations, creating flexible and reusable assets for weapons, enemy behavior, and level layouts.
